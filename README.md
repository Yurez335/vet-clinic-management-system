# Информационная система управления деятельностью ветеринарной клиники

Проект разработан в рамках выпускной квалификационной работы.

## Описание

Система предназначена для автоматизации работы ветеринарной клиники и включает:

* управление владельцами животных;
* ведение карточек пациентов;
* запись на приём;
* фиксацию результатов осмотра;
* просмотр истории лечения;
* формирование аналитических отчётов.

## Архитектура

Система реализована по клиент-серверной архитектуре.

Серверная часть построена на ASP.NET Core и предоставляет API для работы с данными.
Клиентская часть реализована в виде настольного приложения (WPF).
В качестве СУБД используется PostgreSQL.

## Технологии

* ASP.NET Core (.NET 8)
* Entity Framework Core
* PostgreSQL
* WPF
* C#

## Основные модули

* пользователи и роли;
* владельцы животных;
* пациенты;
* запись на приём;
* ведение приёма;
* история посещений;
* отчёты.

## База данных

Система использует реляционную базу данных PostgreSQL.
Основные сущности:

* users
* owners
* pets
* doctors
* appointments
* visits
* services
* appointment_services
